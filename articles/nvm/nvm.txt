Node.js এর ভার্শন সমাচার
মাঝেমধ্যে কোন পুরাতন প্রজেক্ট রান করলে বা গিটহাব থেকে কোন প্রজেক্ট ক্লোন করলে দেখা যায় যে, ঐ প্রজেক্টে Node.js এর যে ভার্শন ব্যবহার করা হয়েছে সেই ভার্শন কম্পিউটারে ইন্সটল না থাকলে প্রজেক্ট রান করা যায় না । এখন আপনি চিন্তা করতে পারেন তাহলে ঐ স্পেসিফিক ভার্শন Node.js এর অফিসিয়াল সাইট থেকে ইন্সটল করে নিলেই তো হয়। ওকে নো প্রব্লেম । একটু পরে দেখলেন আরেকটা প্রজেক্টে অন্য আরেকটা ভার্শন লাগবে তখন কি আবার ঐ স্পেসিফিক ভার্শন Node.js এর অফিসিয়াল সাইট থেকে ইন্সটল করবেন? 

এভাবে Node.js এর অফিসিয়াল সাইট থেকে ভার্শন খুজে খুজে বের করে next next করে ইন্সটল করা খুবই বিরক্তিকর। এর সমাধান হচ্ছে NVM ব্যবহার করা। 

প্রথমে কম্পিউটারে NVM ইন্সটল করে নিতে হবে নিচের লিঙ্ক থেকে।
কম্পিউটারে NVM ঠিকঠাক ইন্সটল হয়েছে কিনা চেক করার জন্যঃ 
nvm -v 
যদি NVM এর ভার্শন দেখতে পান তাহলে ঠিকমতো ইন্সটল হয়েছে, না হলে ঐ স্টেপ আবার ট্রাই করুন। 

আমি ধরে নিচ্ছি আপনি Node.js এর LTS ভার্শন ব্যবহার করছেন। তাহলে কোন ভার্শন ব্যবহার করছেন সেটা দেখতে 
node -v অথবাঃ nvm current
এখন Node.js এর ভার্শন দেখতে পাবেন। 

এখন আপনার প্রয়োজন হচ্ছে Node.js এর 14.20.0 এই ভার্শনে মুভ করা। তাহলে প্রথমে এই ভার্শন NVM এর মাধ্যমে ইন্সটল করে নিতে হবে কম্পিউটারে। ইন্সটল করতেঃ
nvm install 14.20.0
ব্যাস এইটুকু কমান্ড। 
এখন এই ভার্শনে মুভ করতে হলেঃ
nvm use 14.20.0
এটা খুব গুরুত্বপূর্ন যে প্রথমবার ইন্সটল করার পরে ঐ ভার্শনে মুভ করা যাবে।  একটা ভার্শন একবার ইন্সটল করলে চলবে বারবার ইন্সটল করা লাগবে না।

এখন আপনি চাচ্ছেন Node.js এর একবারে latest ভার্শনে মুভ করতে, তাহলেঃ
nvm install latest
এখন এই ভার্শন ব্যবহার করতে পারবেন। ব্যবহার করতে হলেঃ
nvm use latest


এখন যদি চান Node.js এর LTS ভার্শনে মুভ করতে, তাহলেঃ
nvm use lts 
যদি আগে থেকে Node.js ইন্সটল করা না থাকে, তাহলে LTS ভার্শন ইন্সটল করে নিতে হবে। ইন্সটল করতেঃ
nvm install lts

এখন আপনি যদি দেখতে চান কি কি ভার্শন কম্পিউটারে ইন্সটল করা আছে তাহলেঃ
nvm list 
এখন দেখতে পাবেন এবং আপনি যে ভার্শনে আছেন এটার পাশে স্টার (*) চিহ্ন দিয়ে সিলেক্ট করা দেখতে পাবেন । এই লিস্ট দেখে নিয়ে কম্পিউটারে ইন্সটল করা যেকোন ভার্শনে মুভ করতে পারবেন। 
তখন মুভ করতে শুধু nvm use node-version-number দিবেন। উদাহরনঃ nvm use 16.0.0


এখন Node.js এর কি কি ভার্শন আছে সেটা দেখতে চান, তাহলেঃ
nvm list available
এখানে Node.js এর সর্বশেষ ২০টি 
ভার্শন এর লিস্ট দেখাবে যদি আপনার কাঙ্খিত ভার্শন না পান তাহলে যেতে হবে Node.js অফিসিয়াল সাইট এ।


সারাংশঃ এই আলোচনা হচ্ছে Windows 10 এ কিভাবে NVM ব্যবহার করে Node.js এর বিভিন্ন ভার্শনে কিভাবে মুভ করা যায় । 
স্মরন রাখার জন্য সাধারন কমান্ডগুলো আবার বলে দিচ্ছিঃ 

Node.js এর একেবারে latest ভার্শন ইন্সটল করতে nvm install latest আর ব্যবহার করতে nvm use latest ।

Node.js এর LTS ভার্শন ইন্সটল করতে nvm install lts আর ব্যবহার করতে nvm use lts ।

Node.js এর যেকোন স্পেসিফিক ভার্শন ইন্সটল করতে nvm install node-version-number আর ব্যবহার করতে nvm use node-version-number ।
